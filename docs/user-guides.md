# User Guides

## Passenger app guide
- Create account/login.
- Book and track rides.
- Discover restaurants, place food orders, and track delivery.
- Manage wallet, cards, promos, profile, and support tickets.
- FAQ reference: `docs/faq.md#passenger`.

## Driver app guide
- Register and complete onboarding/KYC.
- Toggle availability, accept rides/orders, complete trips.
- Review earnings, payout data, and notifications.
- Use in-app support for incidents.
- FAQ reference: `docs/faq.md#driver`.

## Restaurant owner guide
- Register restaurant and upload verification documents.
- Manage menu/categories, pricing, and availability.
- Process orders and review analytics/performance.
- FAQ reference: `docs/faq.md#restaurant`.

## Admin guide
- Use dashboard for live operations and KPIs.
- Manage users, drivers, restaurants, promotions, support, and risk events.
- Export analytics and audit logs.
- FAQ reference: `docs/faq.md#admin`.
