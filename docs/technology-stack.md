# Technology Stack

## Backend
- Node.js + TypeScript + Express
- Zod validation
- JWT authentication
- Socket.IO
- BullMQ + Redis for jobs

## Frontend/Web
- Next.js (admin and web)
- TypeScript + Tailwind CSS

## Mobile
- Expo + React Native + TypeScript
- Jest + React Native Testing Library

## Infrastructure
- Docker / docker-compose
- GitHub Actions (CI, deploy, CodeQL)
- GHCR image publishing
- Codemagic for mobile artifact workflows

## Third-party integrations
- Stripe-style payment webhook flow
- Twilio (SMS), SendGrid (email), FCM (push)
