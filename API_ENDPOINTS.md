# API Endpoints

Auth, rides, drivers, payments, wallet, KYC, safety, support, merchant, marketplace, admin.

Restaurant backend APIs are available under:
- `/api/restaurants/*` (authentication, profile, documents, menu, variants/add-ons, orders, analytics, payouts, settings, notifications, staff, promotions, reviews, compliance, batch, webhooks)
- `/api/orders/food/*` (public food-order create/track/cancel/rate/refund flows)
- `/api/admin/restaurants/*` (admin restaurant moderation and commission management)

The complete, continuously maintained endpoint catalog is available at:

- [docs/api/endpoints.md](./docs/api/endpoints.md)
- [docs/api/openapi.yaml](./docs/api/openapi.yaml)
